# 📝 To-Do List Application\n\n**A simple, elegant to-do list app with local storage functionality.**\n\n## 🌟 Features\n\n✅ **Add Tasks** - Quickly add new to-do items\n✅ **Mark Complete** - Check off completed tasks\n✅ **Delete Tasks** - Remove individual or all tasks\n✅ **Filter Tasks** - View All, Active, or Completed tasks\n✅ **Priority Levels** - High, Medium, Low priority tasks\n✅ **Local Storage** - All data saved in browser (persists on refresh)\n✅ **Task Stats** - See active and completed count\n✅ **Responsive Design** - Works on mobile and desktop\n✅ **Beautiful UI** - Modern gradient design with animations\n\n## 🚀 Getting Started\n\n1. **Download/Clone the files**\n   - `index.html` - HTML structure\n   - `styles.css` - Styling\n   - `app.js` - JavaScript logic\n\n2. **Open in Browser**\n   ```bash\n   # Simply open index.html in your web browser\n   open index.html\n   ```\n\n## 💾 Local Storage\n\nAll tasks are automatically saved to browser's localStorage:\n- Tasks persist even after closing the browser\n- No server required\n- Automatic save on every action\n\n## 📱 Usage\n\n### Adding a Task\n1. Type task name in the input field\n2. Click \"Add Task\" or press Enter\n3. Task appears at the top of the list\n\n### Completing a Task\n- Click the checkbox next to the task\n- Task text becomes crossed out\n- Move to \"Completed\" filter to view\n\n### Deleting a Task\n- Click \"Delete\" button on any task\n- Or use \"Clear Completed\" to remove all completed tasks\n- Or \"Delete All\" to remove everything\n\n### Filtering Tasks\n- **All** - View all tasks\n- **Active** - View only incomplete tasks\n- **Completed** - View only completed tasks\n\n## 🎨 Customization\n\n### Change Colors\nEdit `styles.css`:\n```css\nbody {\n    background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);\n}\n\n.add-btn {\n    background: #667eea;\n}\n```\n\n### Modify Storage Key\nEdit `app.js`:\n```javascript\nthis.manager = new TodoManager('my-custom-key');\n```\n\n## 📊 Data Structure\n\nEach task is stored as:\n```javascript\n{\n    id: 1621234567890,\n    text: \"Buy groceries\",\n    completed: false,\n    priority: \"high\",\n    createdAt: \"5/13/2026\"\n}\n```\n\n## 🔒 Security\n\n- HTML escaping prevents XSS attacks\n- Input validation\n- No external dependencies\n- Client-side only (no data sent to server)\n\n## 🌐 Browser Compatibility\n\n✅ Chrome/Edge\n✅ Firefox\n✅ Safari\n✅ Opera\n✅ Mobile browsers\n\n## 📝 File Size\n\n- `index.html` - ~1.6 KB\n- `styles.css` - ~5.5 KB\n- `app.js` - ~4.5 KB\n- **Total**: ~11.6 KB\n\n## 🎯 Future Enhancements\n\n- [ ] Edit existing tasks\n- [ ] Due dates\n- [ ] Categories\n- [ ] Subtasks\n- [ ] Dark mode\n- [ ] Export/Import\n- [ ] Cloud sync\n\n---\n\n**Made with ❤️ by Dostonbek Bobokulov**\n"
